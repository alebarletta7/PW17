# Report di sostenibilità – L’Insalata dell’Orto

La pagina presenta una sintesi divulgativa delle iniziative di sostenibilità dell’azienda L’Insalata dell’Orto e permette di accedere rapidamente ai report ufficiali in formato PDF.

---

# Struttura della pagina

La pagina è organizzata nelle seguenti sezioni:

- **Header / Navigation**  
  Barra di navigazione con menu desktop e menu mobile.

- **Hero section**  
  Sezione iniziale con immagine di sfondo e call-to-action.

- **Introduzione**  
  Breve descrizione del contenuto della pagina.

- **Iniziative di sostenibilità**  
  Tre card informative su:
  - energia rinnovabile
  - agricoltura sostenibile
  - filiera controllata

- **Dati principali**  
  Indicatori sintetici relativi alle attività dell'azienda.

- **Download report**  
  Area con link diretti ai report di sostenibilità in formato PDF.

- **Crediti**  
  Informazioni sul project work.

---

# Tecnologie utilizzate

Il progetto è stato realizzato utilizzando:

- **HTML5**
- **CSS3**
- **Flexbox** per la navigazione
- **CSS Grid** per card, statistiche e sezione download
- **Media Queries** per il layout responsive

Il menu mobile è implementato tramite il selettore `:target`,  
senza l'utilizzo di JavaScript.

---

# Responsive design

La pagina è progettata per adattarsi a diverse dimensioni di schermo.

Breakpoints principali:

- **900px** → adattamento card e statistiche  
- **800px** → layout semplificato della tabella download  
- **520px** → attivazione menu mobile

---

# Accessibilità

Sono stati considerati alcuni aspetti di accessibilità:

- attributi **alt** per le immagini
- **aria-label** nella navigazione
- supporto a **prefers-reduced-motion** per utenti sensibili alle animazioni
